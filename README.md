# SoftSell - Software License Marketplace

SoftSell is a responsive, modern marketing website for a fictional software license resale platform.

## Features Implemented

### Core Features
- Responsive single-page design
- Dark/light mode toggle with user preference detection
- Smooth scroll behavior for navigation links
- Animated sections with intersection observer
- Form validation on the contact/lead form

### Section Components
1. **Hero Section**
   - Compelling headline and subheading
   - Clear call-to-action buttons
   - Trusted-by companies display

2. **How It Works**
   - Three-step process with icons
   - Visual explanation of the software resale journey

3. **Why Choose Us**
   - Four value proposition tiles
   - Custom icons for each feature

4. **Customer Testimonials**
   - Two testimonials with user photos
   - Company and role information for each testimonial

5. **Contact Form**
   - Form validation with error handling
   - License type dropdown selection
   - Form submission feedback

### Technical Implementation
- Component-based architecture with React
- Tailwind CSS for styling
- Responsive design for all devices
- SEO optimization with meta tags
- Animated sections using Intersection Observer API
- Context API for theme management

## Design Choices

- **Color Palette**: 
  - Primary blue (#2563EB) for CTAs and primary actions
  - Accent amber (#F59E0B) for highlighting key elements
  - Neutrals for text and backgrounds
  - Dark mode with appropriate contrast

- **Typography**:
  - Inter font family for clean, modern look
  - Clear type hierarchy with appropriate sizing
  - 150% line height for body text, 120% for headings

- **Visual Design**:
  - Subtle dot pattern in the hero background
  - Consistent rounded corners (border-radius)
  - Purposeful white space and padding
  - Subtle shadows for depth
  - Subtle animations on scroll for engagement

- **UX Considerations**:
  - Dark/light mode toggle with user preference detection
  - Smooth scroll behavior for navigation
  - Responsive navigation with mobile toggle
  - Error states in the form with clear feedback
  - Success state after form submission
